# BookReader

A minimal, elegant, and powerful web-based book reader application. Built with modern web technologies and designed for a premium reading experience.

## Features

- 📖 **Multiple Book Samples**: Comes pre-loaded with classics like Alice in Wonderland, Sherlock Holmes, and Pride and Prejudice.
- 📁 **File Support**: Open your own `.txt` or `.pdf` files directly in the browser.
- 🎨 **Customizable Experience**: Adjust font size, line height, and toggle between light and dark modes.
- 📱 **PWA Ready**: Installable on your device for an app-like experience, powered by Progressier.
- ⚡ **Performance**: Fast, responsive, and works across all modern browsers.

## Technologies Used

- HTML5
- CSS3 (Vanilla)
- JavaScript (ES6+)
- [PDF.js](https://mozilla.github.io/pdf.js/) for PDF rendering
- [Progressier](https://progressier.com/) for PWA capabilities

## Installation

Since this is a PWA, you can simply open the app in your browser and select "Add to Home Screen" or "Install" from the browser menu.

## Development

To run locally, simply open `index.html` in any modern web browser or use a local development server like `Live Server` in VS Code.
